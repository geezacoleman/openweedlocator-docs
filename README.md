# OpenWeedLocator Documentation

[![Documentation Status](https://readthedocs.org/projects/openweedlocator/badge/?version=latest)](https://docs.openweedlocator.org/en/latest/?badge=latest)

Documentation source for the [OpenWeedLocator (OWL)](https://github.com/geezacoleman/OpenWeedLocator) project — an open-source, camera-based weed detection system for site-specific weed control.

The live documentation is at **[docs.openweedlocator.org](https://docs.openweedlocator.org)**.

Built with [Sphinx](https://www.sphinx-doc.org/), [MyST Parser](https://myst-parser.readthedocs.io/), and the [Furo](https://pradyunsg.me/furo/) theme. Hosted on [Read the Docs](https://readthedocs.org/).
